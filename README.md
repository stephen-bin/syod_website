# The Spiritual Youth on a Date (Official Website)

Landing page for the book *The Spiritual Youth on a Date* by Stephen Npoandan Binaansim.

## Features

- **Modern Landing Page**: Responsive design, animations, and "The Modern Apostle" branding.
- **Shopping Cart**: Add books to cart, persistence via LocalStorage.
- **WhatsApp Checkout**: Order directly via WhatsApp with a pre-filled message.
- **Paystack Integration**: Direct link to secure payment page.
- **Event Registration**: Request access via WhatsApp.

## Deployment

### Static Hosting (Recommended)

This site is built with pure HTML, CSS, and Client-Side JavaScript.
It can be hosted on **GitHub Pages**, **Netlify**, or **Vercel** with zero configuration.

1. Push code to GitHub.
2. Enable GitHub Pages in Settings.
3. Done!

## Project Structure

- `index.html`: Main entry point.
- `css/style.css`: All styling.
- `js/`: Application logic.
- `data/`: Static data (events).
